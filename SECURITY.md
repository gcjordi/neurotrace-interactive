# Security Policy

## Supported Versions

NeuroTrace Interactive is currently maintained as a public demonstration project.

Security updates are applied to the latest version available in the `main` branch.

## Reporting a Vulnerability

If you discover a security issue, please use GitHub's private vulnerability reporting feature when available.

When reporting a vulnerability, please include:

- A description of the issue
- Steps to reproduce the problem
- Potential impact
- Any suggested mitigation

We will review all legitimate reports and make reasonable efforts to address confirmed security issues.

## Scope

NeuroTrace Interactive is an educational and demonstrative web application.

The project:

- Does not process sensitive personal data
- Does not require user accounts
- Does not store user information
- Does not use external AI APIs
- Performs all processing locally in the browser

## Disclaimer

NeuroTrace Interactive is an illustrative AI interpretability demo intended for educational and showcase purposes.

It does **not** provide:

- Real model introspection
- Security auditing
- Certified AI safety assessments
- Access to proprietary model internals

All visualizations are illustrative representations designed to improve understanding of AI concepts and response dynamics.
